## 🔍 What is soc-monthly-consumption-report?

This is an Excel template designed to help you track monthly consumption of office utilities, supplies, and manpower. It works by letting you enter usage data into predefined fields. The spreadsheet then performs automatic calculations to summarize your data in an easy-to-understand dashboard. This can help you manage costs and understand how your office resources are being used each month.

## 📋 Key Features

- Track consumption of utilities like water, electricity, and tea/coffee.
- Monitor supplies such as stationery, medical supplies, and food.
- Record manpower use including attendance and payroll details.
- Automated calculations to summarize costs and usage.
- Dashboard view to quickly see trends and totals.
- Supports managing multiple company locations in one file.
- Suitable for office management, housekeeping, security, and other services.

## 💻 System Requirements

- Windows 7 or later.
- Microsoft Excel 2016 or newer with macros enabled.
- At least 4 GB of RAM recommended for smooth operation.
- Basic familiarity with Excel for data entry.

### 2. Open the File in Excel

- Double-click the downloaded file.
- If Excel displays a security warning about macros, click **Enable Content**. The calculations and dashboard require macros to function.
- Familiarize yourself with the multiple sheets inside the file. Typical sheets may include:
  - Data entry forms for utilities, supplies, and manpower.
  - Summary dashboard that updates automatically.
  - Instructions or notes on usage.

### 3. Enter Your Data

- Use the first sheet to enter monthly dates and units consumed.
- Fill in supplies used each month as per your office needs.

- Avoid deleting or altering the formulas in the calculation or dashboard sheets.

### 4. View the Dashboard

- Switch to the dashboard tab to see organized summaries.
- This tab shows totals, averages, comparisons, and cost tracking.
- Use this information to adjust orders or manage budgets better.

## ⚙️ How to Use the Template

## 🛠️ Troubleshooting Common Issues

- **Macros not working:** Ensure you have enabled macros when opening the file.
- **File won’t open:** Make sure you are using Microsoft Excel, not another tool.
- **Data not updating:** Check that you entered values into proper fields only.
- **Dashboard blank or incorrect:** Avoid modifying cells in the dashboard sheet manually.

## 📝 Additional Tips

- Back up the file before making major changes.
- Use one file per month or combine all months in one workbook.
- Customize categories by renaming headers if your office tracks other supplies.
- Use the sorting and filtering options in Excel to find specific data points.
